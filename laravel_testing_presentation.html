🏗️ Test Piramidi

📚 Test Piramidi Nedir?

Test Piramidi, yazılım testlerini farklı granülaritede gruplandırmak için kullanılan bir metafordur. Mike Cohn tarafından "Succeeding with Agile" kitabında ortaya atılmıştır.

Ana Prensipler:

  • Farklı granülaritede testler yazın
  • Yüksek seviyeye çıktıkça daha az test yazın
  • Piramit şeklini koruyun - test ice-cream cone olmasın!
E2E Tests (Az)
End-to-End • Sistem • Kapsamlı
Feature Tests (Orta)
Integration • API • HTTP
Unit Tests (Çok)
Hızlı • İzole • Küçük

✨ Best Practices

🎯 Test Yazım Kuralları

  • AAA Pattern: Arrange, Act, Assert
  • Açıklayıcı isimler: test_user_can_create_post
  • Tek sorumluluk: Bir test, bir şey test etmeli
  • Bağımsızlık: Testler birbirini etkilememeli

🚀 Performans İpuçları

  • In-Memory DB: SQLite kullanın
  • RefreshDatabase: Trait'ini kullanın
  • Factory'ler: Gerçek data yerine
  • Mock'lar: Harici servisleri taklit edin

📝 Pratik Örnekler - Model Testing

👤 User Model Test

class UserTest extends TestCase { use RefreshDatabase; public function test_user_can_be_created() { $userData = [ 'name' => 'John Doe', 'email' => 'john@example.com', 'password' => bcrypt('password123') ]; $user = User::create($userData); $this->assertInstanceOf(User::class, $user); $this->assertEquals('John Doe', $user->name); $this->assertDatabaseHas('users', [ 'email' => 'john@example.com' ]); } public function test_user_email_must_be_unique() { User::factory()->create(['email' => 'test@test.com']); $this->expectException(QueryException::class); User::factory()->create(['email' => 'test@test.com']); } }

🔐 Authentication & Authorization

class PostControllerTest extends TestCase { use RefreshDatabase; public function test_guest_cannot_create_post() { $response = $this->post('/posts', [ 'title' => 'Test Post', 'content' => 'Test content' ]); $response->assertRedirect('/login'); } public function test_user_can_create_post() { $user = User::factory()->create(); $response = $this->actingAs($user) ->post('/posts', [ 'title' => 'My First Post', 'content' => 'This is my first post content.' ]); $response->assertStatus(201); $this->assertDatabaseHas('posts', [ 'title' => 'My First Post', 'user_id' => $user->id ]); } }
